# PKICertificateMP
PKI Certificate MP for SCOM next version

Years ago Raphael Burri created a PKI Certificate monitoring MP for SCOM.
The repo in the original location does not get any updates and changes.
We were assigned years ago to be able to make changes, but the approval process in Github stops this.
Hereby a repo for newer versions of the same SCOM MP.

We will continue versioning here and upload the documentation in the zip file contining the files.
Please add Issue items for questions or bugs or requests. We can not always react quickly to those, but we are incorporating a number already from the old Repo.
